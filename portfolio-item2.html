Uber Web Appliction

Utilizing react, react-routers, tailwind styled components, Mapbox, Firebase

Login page for the Uber application

Utilizing Firebase to access Google Login functionalities

Landing page for the application, seperated into two parts including a mapbox and the buttons section. The buttons section includes a logo, logout button, and three clickable buttons.

Clicking on the "Ride" option will redirect the user the input page where the user is able to choose pickup location and dropoff location.

Upon selecting "Confirm Locations", the fare for the rides will be computed by using the Mapbox API to determine the travel time between the two coordinates. The Mapbox display will then highlight the two coordinates and adjust the display to center them. The application will present all available ride alternatives in a scrollable list format, allowing users to choose an option. Each ride option will have a unique image and price multiplier associated with it.

Choosing the "Wheel" option from the homepage will have similar functionalities as the "Ride option".

Choosing the Reserve option will redirect the user to a reserve page with a image section and a explanation section with icons and text. At the button there is a button that allows users to "Reserve a ride".
